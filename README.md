# Chip-to-Chip (C2C) Flow Prediction

This project presents a chip-to-chip (C2C) machine learning framework for accelerating CFD-style flow-field prediction. Instead of treating the full flow field as a single global mapping problem, the C2C architecture models spatial evolution progressively through local chips, making the learning process more consistent with underlying physical behaviour.

The project was developed to address limitations of traditional machine learning models in handling complex simulation data across varying geometries, resolutions, and operating conditions. In addition to model development, the workflow includes simulation data preprocessing, structured dataset generation, experiment management, and analytics on large-scale scientific data.

This repository reflects my work at the interface of physics-based simulation, machine learning, and data-processing system design.
